This folder contains all data and r script for model validation for the ELM_SPRUCE model (Ricciuto et al. 2020)

Ricciuto, D., X. Xu, X. Shi, Y. Wang, X. Song, C. W. Schadt, N. A. Griffiths, J. Mao, W. Jeff., P. Thornton, J. Chanton, J. K. Keller, S. Bridgham, J. Gutknecht, S. Sebestyen, A. C. Finzi, R. K. Kolka, and P. J. Hanson. 2020. An Integrative Model for Soil Biogeochemistry and Methane Processes: I. Model Structure and Sensitivity Analysis. Journal of Geophysical Research-Biogeosciences minor revision.

The model code is archived at: Xiaofeng Xu, & Daniel Ricciuto. (2020, March 31). dmricciuto/CLM_SPRUCE: Release of CLM_SPRUCE (Version v1.0.0). Zenodo. http://doi.org/10.5281/zenodo.3733924
